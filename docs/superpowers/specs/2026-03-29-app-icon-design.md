# Recolly アプリアイコン — デザイン仕様書

## 1. 概要

Recollyアプリのアイコンをリニューアルする。既存の紫グラデーションのfaviconを、ブランドコンセプトに沿ったシンプルで洗練されたデザインに置き換える。

## 2. デザインコンセプト

### コンセプト
「落ち着き・洗練された大人の記録ツール」

### 方向性
- セリフ体の「R」（Recollyの頭文字）をメインモチーフとする
- Rの脚（右下に伸びる斜線部分）に3色のカラーラインを添え、「複数ジャンルをまとめて管理できる」コンセプトをさりげなく表現する
- 色は密集させず、最小限の要素で構成する

### 却下した方向性
- グラデーション型（Instagram風）→ 却下
- ジャンルカラー6色全部を使う → 密集感が出るため却下
- 抽象・幾何学型（円や図形の集合）→ 密集感が出るため却下

## 3. デザイン仕様

### フォント
- **Georgia Bold (700)**
- SVG内では `font-family="Georgia, 'Times New Roman', serif"`

### カラー

| 要素 | カラーコード | 用途 |
|------|------------|------|
| 文字（R） | #2c2c2c | メインの文字色（既存のデザイントークン `--color-text` と同じ） |
| 背景 | #ffffff | 白背景（角丸四角） |
| ライン1 | #5e548e | 紫（既存の `--color-movie`） |
| ライン2 | #6b9080 | グリーン（既存の `--color-game`） |
| ライン3 | #c4956a | ブラウン（既存の `--color-book`） |

### 3色ラインの配置
- Rの脚（右下に伸びる部分）に沿って3本の短いラインを配置
- 各ラインは脚の角度に合わせて回転（約-50度）
- 長さは段階的に短くなる（22px → 18px → 14px、viewBox 140x140基準）
- 太さは 3.5px（viewBox 140x140基準）
- 角丸端（rx=1.75）

### 背景
- 白背景（#ffffff）の角丸四角
- 角丸の半径: viewBox 140x140基準で rx=28（比率20%）

## 4. 生成するファイル

| ファイル | サイズ | 用途 |
|---------|-------|------|
| `frontend/public/favicon.svg` | SVG | ブラウザのfavicon |
| `frontend/public/icons/icon-192x192.png` | 192×192px | PWAアイコン（小） |
| `frontend/public/icons/icon-512x512.png` | 512×512px | PWAアイコン（大） |
| `frontend/public/icons/apple-touch-icon-180x180.png` | 180×180px | Apple Touch Icon |

### favicon.svg の仕様
- 既存のグラデーションデザインを新デザインに置き換える
- viewBox="0 0 140 140"
- Rを中央配置（text-anchor="middle", dominant-baseline="central", transform で位置調整）

### PNG生成について
- SVGからPNGへの変換が必要
- 変換手段は実装プランで決定する（sharp、canvas、手動エクスポート等）
