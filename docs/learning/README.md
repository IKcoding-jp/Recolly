# 技術学習ノート

## このディレクトリの目的

Recollyで使った技術について、初学者向けの解説ノートを蓄積する場所です。

AI支援開発では、コードは動くが「なぜ動くか説明できない」状態（理解負債）が蓄積しやすくなります。このディレクトリは、その理解負債を「返済」するための仕組みです。

各ノートは「この技術は何か」「Recollyでどう使っているか」「ハマりやすいポイント」を初学者向けに解説します。

## いつ書くか

- 新しい技術やライブラリを導入したとき
- 技術の仕組みについて疑問を持ったとき
- 開発中に「これは理解しておいた方がいい」と感じたとき

## ファイル命名規則

ケバブケースの英語で、技術名や概念名をそのまま使う。

例:
- `jwt-authentication.md`
- `n-plus-1-query.md`
- `react-hooks.md`
- `docker-compose.md`
