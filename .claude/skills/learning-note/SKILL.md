# 技術学習ノート

## このスキルの目的

Recollyで使った技術について、初学者向けの解説ノートを作成する。
「なぜ動くか」「何をしているか」を自分の言葉で説明できる状態を目指す。

## ルール

### ファイル作成

- 保存先: `docs/learning/`
- ファイル名: ケバブケースの英語
  - 例: `jwt-authentication.md`, `n-plus-1-query.md`, `react-hooks.md`
- テンプレート: `assets/template.md` に従う

### 必須の内容

- 説明は必ずプログラミング初学者向け。身近な例えを積極的に使う
- Recollyでの具体的な使用箇所（ファイルパス等）を含める
- 対応するADRがあればリンクする
- 「注意点・ハマりやすいポイント」は開発中に実際に遭遇した問題を記載

### 書くタイミング

- 新技術を導入した直後（comprehension-guardからの提案時）
- ユーザーが技術について質問したとき（質問への回答をノート化）
- 開発中に「これ理解しておいた方がいいな」と判断したとき

### 書かなくてよい場合

- 既に学習ノートが存在する技術
- 一般常識レベルの概念（HTML, CSS の基本等）
